Frame

A frame is a part of a web page that can display content independently of the rest of the page. Frames are created using the <iframe> tag in HTML. An iframe allows you to embed another HTML document within the current document. This can be useful for displaying content from another website, such as a video, map, or social media feed, without having to leave your own site.

Iframe Attributes

  • src: Specifies the URL of the page to embed
  • width: Specifies the width of the iframe
  • height: Specifies the height of the iframe
  • title: Provides a title for the iframe (used for accessibility)
  • frameborder: Specifies whether to display a border around the iframe (deprecated in HTML5)
  • allowfullscreen: Allows the iframe content to be displayed in fullscreen mode

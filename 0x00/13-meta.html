Meta Tags

the <meta> tag is used to provide metadata about the HTML document. metadata is data about the data. The <meta> tag is used to specify page description, keywords, author of the document and other meta data. The metadata can be used by browsers(how to display content or reload page) and search engines or other web services.

Meta Tag Attributes

  • name: Specifies the name of the meta property
  • content: Specifies the content of the meta property
  • charset: Specifies the character encoding for the HTML document
  • http-equiv: Provides an HTTP header for the information/value
